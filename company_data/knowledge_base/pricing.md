FlowForge offers flexible pricing plans designed for growing teams, agencies, and product organizations that need structured workflow management and automation capabilities.

All plans are billed per user per month and can be billed monthly or annually. Annual billing provides a 15% discount.

---

## Starter — $19/user/month

The Starter plan is designed for small teams and early-stage startups that need a structured way to manage tasks and workflows without complex administrative controls.

### Best For
Small teams, founders, and startups beginning to formalize internal processes.

### Included Features

- Up to 10 users
- Unlimited task boards
- Basic workflow automation
- Task comments and file attachments
- Basic AI task summarization
- Slack integration
- 5GB storage per workspace
- Standard email support
- Community knowledge base access

### Limits

- Maximum 10 seats
- Up to 50 automation rules
- Basic analytics dashboard
- No advanced permission controls
- No audit logs

---

## Growth — $49/user/month

The Growth plan is designed for agencies and product teams that need more advanced workflow automation, analytics, and team management features.

### Best For
Growing teams, agencies, and small software companies managing multiple projects.

### Included Features

Everything in Starter, plus:

- Unlimited users
- Advanced workflow automation
- AI task summarization with contextual insights
- Advanced analytics dashboard
- Role-based team permissions
- GitHub integration
- Google Workspace integration
- Zapier integration
- 100GB storage per workspace
- Priority email support
- Automation usage analytics

### Limits

- Up to 500 automation rules
- Limited audit logs (30 days)
- Standard API rate limits

---

## Enterprise — Custom Pricing

The Enterprise plan is built for organizations that require advanced security, compliance, and scalable automation infrastructure.

### Best For
Large product teams, enterprise agencies, and companies managing complex automation pipelines.

### Included Features

Everything in Growth, plus:

- Unlimited automation rules
- Dedicated workspace infrastructure
- Advanced security controls
- Single Sign-On (SSO)
- Full audit logging
- Compliance support
- Custom API rate limits
- Dedicated customer success manager
- SLA-backed uptime guarantees
- Custom integrations support
- Data retention controls
- Unlimited storage

### Support Level

Enterprise customers receive:

- Dedicated onboarding support
- 24/7 priority support
- Architecture guidance
- Quarterly platform reviews
