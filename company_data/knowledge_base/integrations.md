FlowForge integrates with popular tools used by modern product teams. Integrations allow teams to connect workflows, automate updates, and synchronize data across platforms.

---

## Slack Integration

The Slack integration allows FlowForge to send notifications and updates directly into Slack channels.

### Features

- Task status notifications
- Automation alerts
- Project milestone updates
- Team mentions and reminders

Teams can configure which events trigger Slack messages.

---

## GitHub Integration

The GitHub integration connects development workflows to FlowForge task management.

### Features

- Link pull requests to tasks
- Automatically update task status when pull requests are merged
- Track code deployment progress
- Associate commits with project tasks

This integration allows product and engineering teams to keep project planning aligned with development work.

---

## Google Workspace Integration

FlowForge integrates with Google Workspace to streamline collaboration.

### Features

- Attach Google Docs to tasks
- Connect Google Drive files
- Use Google authentication
- Sync calendar events with project deadlines

This integration simplifies collaboration for teams already using Google Workspace tools.

---

## Zapier Integration

Zapier allows FlowForge to connect with thousands of external applications.

### Examples of Automation

- Create tasks from form submissions
- Sync CRM updates to project workflows
- Trigger workflows from external events
- Send automated notifications across systems

Zapier enables non-technical users to build automation pipelines without coding.

---

## Notion Integration

The Notion integration helps teams synchronize documentation and project execution.

### Features

- Link Notion pages to tasks
- Sync documentation with project workflows
- Reference product specifications inside task boards

This integration ensures that documentation and operational workflows remain connected.
